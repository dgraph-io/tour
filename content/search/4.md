The `exact` index lets you use inequality operators (`lt`, `le`, `gt`, `ge`) on
strings in filters. Add the `exact` index
[to the string indexes]({{< relref "search/1" >}}), then try the query.

The `hash` index provides fast `eq` filtering on strings.
