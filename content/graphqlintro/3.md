GraphQL mutations load data into a Dgraph database.

Click **Run** to load the small dataset used in the upcoming lessons.

This tutorial uses an `xid` field to identify nodes for consistency across
mutations and queries. The `id` field with scalar type `ID` would also work,
but assigned IDs vary across instances.
